How to use the dashboard?

it's a ROS-based dashboard that controls turtlebot, it contains 3 parts:
1- the map : as the robot moves around the place, it will capture the environment via the laser sensor and draw a map based on it
2- the video display : it will display what the camera captures
3- the controller : it has 4 buttons to move the robot : forward, bachward, right, and left and one button to stop the robot
